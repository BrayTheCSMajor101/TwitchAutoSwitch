Twitch Auto Switch

A lightweight Twitch automation script that detects when a stream ends and auto-redirects to another live channel.  
Built with **plain JavaScript** using DOM detection and interval checks.  

Features
- Detects when a Twitch stream has ended  
- Redirects to a live followed channel (if available)  
- Falls back to recommended streams on homepage  
- Fully client-side, no dependencies  

How to Use
1. Copy the script from `twitch-auto-switch.js`
2. Paste it into your browser console **while on Twitch**  
   (or use it in a custom extension / userscript manager like Tampermonkey)  

Tech
- JavaScript  
- DOM APIs  
- Browser automation concepts
